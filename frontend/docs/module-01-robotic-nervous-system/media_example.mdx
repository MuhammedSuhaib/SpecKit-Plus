# Media Example - Images and Videos

## Adding Images

You can add images to your chapters using standard Markdown syntax:

![Docusaurus Logo](/img/docusaurus.png)

Or using JSX for more control:

<img src="/img/undraw_docusaurus_mountain.svg" alt="Docusaurus" style={{width: '50%', display: 'block', margin: 'auto'}} />

## Adding Videos

You can embed videos using HTML5 video tags:

<video width="100%" height="auto" controls>
  <source src="/sample-video.mp4" type="video/mp4" />
  Your browser does not support the video tag.
</video>

## Best Practices

1. Store images in the `/static/img/` directory
2. Store videos in the `/static/` directory or use external hosting
3. Use appropriate file formats (JPG/PNG/WebP for images, MP4/WebM for videos)
4. Optimize file sizes for web delivery
5. Always include alt text for accessibility
